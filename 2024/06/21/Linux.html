Linux

This update is for June, July & August. I’ve done no significant gamedev work in the last month and some other tasks have come up that mean nothing is likely to be done until mid-August (at the earliest). So no change.

Ok, a little change. My desktop is now Linux, to be exact Ubuntu 24.04 (it has a numbat for a mascot!). I’ve used Windows for over 25 years now, but before that my home computer was Linux going back to university. Linux still has some annoyances (I guess all OSes do), but it has moved on a long way, and a lot is better than I remember, especially in the ecosystem of programs available. You can even play decent comercial games on it now - what a world we live in.
